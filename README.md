# Vineeta19-prog

This repository contains a simple web project built with **HTML**, **CSS**, and **JavaScript**. It demonstrates basic front-end development practices and includes an image asset.

---

## 📂 Project Structure

- **index.html**  
  The main HTML file that serves as the entry point of the project. It defines the structure of the webpage.

- **index.css**  
  Stylesheet for the project. It contains all the CSS rules used to style the HTML elements.

- **index.js**  
  JavaScript file that adds interactivity and dynamic behavior to the webpage.

- **pic.jpg**  
  Image asset used within the project (e.g., for display on the webpage).
